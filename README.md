# Report

Thư mục `Report` chứa phần tài liệu báo cáo của đồ án.

Hiện tại gồm các thành phần chính:

- `IOT_Application.tex`: mã nguồn LaTeX của báo cáo;
- `README.md`: mô tả nhanh về thư mục tài liệu.

Mục đích của nhánh `main` là lưu riêng phần báo cáo, tách biệt với firmware `Receiver` và `Sender`, để:

- dễ quản lý phiên bản tài liệu;
- dễ chia sẻ cho người chỉ cần phần báo cáo;
- tránh làm lẫn lịch sử mã nguồn firmware với tài liệu viết.
